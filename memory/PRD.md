# ROSHKA STUDIO - Product Requirements Document

## Original Problem Statement
Create a professional website for ROSHKA STUDIO - a creative web studio based in Silver Spring, MD specializing in modern website design. Requirements include dark premium theme (black + gold), glassmorphism elements, contact form with Google Sheets and email notification integrations.

## Architecture
- **Frontend**: React 19 with Tailwind CSS, Framer Motion for animations
- **Backend**: FastAPI with Python
- **Database**: MongoDB for inquiry storage
- **Email**: Gmail SMTP for notifications
- **Google Sheets**: Service account integration (Spreadsheet ID: 1qM6Av8ub72NTFTsAXUzcWvRWgPvk7oZc_RO9-zLG4sQ)
